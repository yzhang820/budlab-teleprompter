# 透明提词器使用说明

打开 `index.html` 就可以使用。

## 常用操作

- 在底部编辑框里修改自我介绍内容，会自动保存到本机浏览器。
- 点“开始”后提词会自动滚动；拖动进度条可以手动调整位置。
- “演讲模式”会隐藏控制面板，只保留提词文字。
- 可调整速度、字号、行距、文字透明度和文字宽度。
- 快捷键：
  - 空格：开始/暂停
  - E：展开/收起编辑
  - H 或 Esc：隐藏/显示面板
  - 上/下方向键：调慢/调快
  - R 或 Home：回到开头

## 覆盖在视频工具上

这个页面本身使用透明背景，文字区域也尽量不遮挡视频内容。需要注意：普通浏览器窗口通常不是真正透明窗口，所以系统可能仍会显示一个普通窗口背景。

想要面试时更自然，建议：

- 把浏览器窗口缩窄，放在摄像头附近。
- 进入“演讲模式”，只保留文字。
- 把文字透明度调低一点，字号调到刚好能看清。
- 如果你使用 OBS、虚拟摄像头或支持透明网页源的工具，可以把这个 HTML 作为透明网页源使用，效果会更接近真正覆盖。
